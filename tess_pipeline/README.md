# TESS Exoplanet Detection Pipeline

This pipeline automates the end-to-end detection of exoplanet transit signals in photometric data from NASA's Transiting Exoplanet Survey Satellite (TESS). Starting from a sky-cone query to the MAST archive, it retrieves two-minute cadence light curves for dwarf stars (log g ≥ 4.0) brighter than TESS magnitude 13, cleans and normalises the flux time series, folds candidate periods identified by a Box Least Squares (BLS) periodogram, and produces publication-quality diagnostic plots alongside a structured results catalogue. All configurable parameters — search geometry, magnitude cuts, API endpoints, and directory layout — are centralised in `src/config.py`, making it straightforward to retarget the pipeline to any region of sky or adjust quality thresholds without touching pipeline logic.
