# TESS Exoplanet Detection Pipeline

This repository contains an end-to-end pipeline designed to query, clean, analyze, and export target stars from the **TESS Input Catalog (TIC)** for exoplanet transit searches, focusing on the southern polar cap region.

## 🛠️ Pipeline Components & Workflow

### 1. Configuration & Utilities
* **`src/config.py`**: Centralizes all settings such as directory paths, search coordinates (Declination centering, Search radius), MAST API settings, and target quality cuts (e.g., Dwarf selection constraints: $T_{mag} < 13.0$, $\log g \ge 4.0$).
* **`src/utils.py`**: Manages logging infrastructure and ensures directories exist.

### 2. Data Download (`src/download.py`)
* Queries the MAST archive for two-minute cadence light curve targets within the specified sky-cone.
* Utilizes `astroquery.mast.Catalogs.query_region()` as the primary path.
* Automatically falls back to direct paginated REST API calls querying 6 Right Ascension (RA) strips tiling the polar cap.
* Caches raw data in `data/raw/tic_southern_polar.csv`.

### 3. Data Cleaning & Imputation (`src/clean.py`)
* Drops columns with $\ge 99\%$ null values and hardcoded irrelevant columns, exporting a diagnostic `data/clean/null_report.csv`.
* Casts remaining TIC columns to correct data types.
* Computes the Gaia $BP - RP$ color index.
* Fits a polynomial Ridge regression model (default degree = 3) to calibrate and predict $T_{eff}$ using the Gaia color index.
* Imputes missing $T_{eff}$ values via the fitted calibration model.
* Featurizes the dataset (adds estimated spectral types `SpType_est`, brightness tiers `Tmag_tier`, proper motion, and distance).
* Evaluates stars and flags suitable dwarfs ($T_{mag} < 13.0$, $\log g \ge 4.0$, non-white dwarf) under a `prime_target` boolean flag.
* Saves the cleaned catalog as a compressed `data/clean/tic_clean.parquet`.

### 4. Exploratory Data Analysis (`src/eda_catalog.py`)
Generates 10 publication-quality diagnostic plots in `plots/` based on the cleaned dataset:
* **`fig_01_null_rates.png`**: Column fill rates.
* **`fig_02_classification.png`**: Object classifications (object type, luminosity class, white dwarf flags).
* **`fig_03_tmag.png`**: TESS magnitude distributions.
* **`fig_04_teff.png`**: Temperature and spectral type distribution.
* **`fig_05_logg.png`**: Surface gravity ($\log g$) and dwarf vs. giant categorization.
* **`fig_06_hr_proxy.png`**: HR/Kiel Diagram proxy (effective temperature vs. surface gravity).
* **`fig_07_cmd.png`**: Color-magnitude diagram (2D histogram).
* **`fig_08_distance.png`**: Distance distributions.
* **`fig_09_sky.png`**: Footprint of targets in sky coordinates.
* **`fig_10_funnel.png`**: Target selection funnel outlining how filters narrow down the candidates.

### 5. Target Exportation (`src/export_targets.py`)
* Filters the cleaned catalog down to prime targets (`prime_target == True`).
* Sorts them by $T_{mag}$ (brightest first).
* Selects a canonical set of columns (identifiers, photometry, physical properties, cross-match IDs) and exports them to `data/clean/prime_targets.csv`.

## 🚀 How to Run

Install dependencies:
```bash
pip install -r requirements.txt
```

### Complete Workflow:
To run the full pipeline:
1. **Download Raw Data**:
   ```bash
   python -c "from src.config import CFG; from src.download import download_tic_catalog; download_tic_catalog(CFG)"
   ```
2. **Clean & Impute Data**:
   ```bash
   python -m src.clean
   ```
3. **Generate EDA Plots**:
   ```bash
   python -m src.eda_catalog
   ```
4. **Export Prime Targets**:
   ```bash
   python -m src.export_targets
   ```
